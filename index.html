Archaic CPU Emulators

The application pretty much does what it says it does.... emulates CPUs

The CPUs are emulated as in the assembly code is compiled and executed on a virtual CPU with a virtual amount of memory.

The assembly code is on the left for the CPU you are targeting, and then you can see the emulation running your code on the right side of the screen.
